Single Page App - Vue.js

Här är mitt projektarbete för kursen "JavaScriptbaserad webbutveckling". Jag har använt Vue.js (cdn) tillsammans med axios för att bygga frontend och konsumera en webbtjänst, som jag också skapat. Jag byggde en server i Node.js med express som publicerades på Heroku. Till denna hör en MongoDB-databas som jag skapade på mlab. Slutligen gjorde jag också en sida för administratörer. Den är byggd på ren JavaScript, html, css och ajax. Från adminsidan finns möjlighet att manipulera innehållet som visas på Vue-sidan med get, put, post och delete-funktioner till webbtjänsten. Notera att det kan ta några sekunder för servern att vakna till.

Tekniker: JavaScript, Vue.js, MongoDB, No-SQL, SASS, CSS, gulp, node.js
